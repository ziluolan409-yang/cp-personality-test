## 🚀 部署到 GitHub Pages（只需做一次）

### 第1步：创建 GitHub 仓库
1. 打开 https://github.com/new
2. Repository name 填：`cp-personality-test`
3. 选 **Public**（公开）
4. **不要**勾选任何初始化选项（README、.gitignore 等）
5. 点击 **Create repository**

### 第2步：推送代码
在终端中运行：
```bash
cd "d:\EP的CC"
git remote add origin https://github.com/ziluolan409-yang/cp-personality-test.git
git push -u origin main
```

### 第3步：开启 GitHub Pages
1. 进入仓库 → **Settings** → **Pages**
2. **Source** 选 `Deploy from a branch`
3. **Branch** 选 `main`，文件夹选 `/ (root)`
4. 点击 **Save**
5. 等待 1-2 分钟，页面 URL 会显示在 Settings → Pages 顶部
