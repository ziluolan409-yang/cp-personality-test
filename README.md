# 嗑CP人格属性测试

一个好玩的性格测试，测测你是哪种嗑学家！

## 在线体验

👉 **[开始测试](https://用户名.github.io/cp-personality-test/cp-personality-test.html)**

（请将 `用户名` 替换为你的 GitHub 用户名）

## 本地运行

直接用浏览器打开 `cp-personality-test.html` 即可。

## 查看使用量

1. **页面底部计数器** — 显示已完成测试的人数
2. **GitHub 流量分析** — 进入仓库 → Insights → Traffic，查看：
   - Page views（页面浏览量）
   - Unique visitors（独立访客）
   - Referring sites（来源网站）
   - Popular content（热门内容）

## 部署到 GitHub Pages

1. 创建 GitHub 仓库（如 `cp-personality-test`）
2. 推送代码到 GitHub
3. 进入仓库 Settings → Pages → Source 选 `main` 分支 → Save
4. 等待几分钟即可通过 `https://用户名.github.io/cp-personality-test/cp-personality-test.html` 访问
